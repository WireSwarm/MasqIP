# Global
Tu dois faire une icone selon la description suivante : 
L’icône représente un symbole en forme d’arborescence :

Une tige verticale part du haut vers un nœud central.

De ce nœud, deux branches diagonales s’étendent vers le bas gauche et bas droite.

Chacune des branches se termine par un segment par un disque pour la branche pointant le haut. Et des carrés pour les 2 branches allant du bas bas.

Tu peux faire ce logo en svg stp.

Ce logo évoque souvent une idée de connexion, hiérarchie, réseau ou structure arborescente (comme dans les graphes, les arbres Git, ou les schémas de dépendances).
